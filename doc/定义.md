### 类和函数

类定义：

```
Class {
    a = 1;

    UPPER_IS_CONST = 1;   // ALL upper will be const.  Implicit

    func1() {

    }   

    // no type param
    func2(p1) {

    }

    // type suggest param
    func3(Int: p2) {

    }

}

```

函数，函数指针， lambda定义： 

```

myFunc() {
    return 1;
}

myFuncPointer = myFunc;

b = myFuncPointer();    // b = 1

// var = lambda
l1 = a, b => {
    return a + b;
}

c = l1(1,2);   // 3

// function filter
filter(list, Consumer<Int>: callback) {
    callback(1);
}

// call function filter

filter(null, i => {a = i + 1;});

```
