- 类粉饰工具

```
ResponseObject {
    code = -1;
}

response => (c) {
    return ResponseObject(code=c);
}

@ext response
ResponseCode {
    OK = 1;
    Fail = 2;
    BadRequest = 3;
}


x = ResponseCode.OK.response();    // x = ResponseObject(code = 1)

```

- 类型建议工具

```
response => (:Int c) {
    return ResponseObject(code=c);
}
```

这里的`:Int` 表示建议传参是一个Int, 但是不强制要求是一个Int,  符合这个接口即可。 



- 不允许重载， 允许参数名指定参数。
